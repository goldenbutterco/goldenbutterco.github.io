
# Luxury Bakery Website Design

This is a code bundle for Luxury Bakery Website Design. The original project is available at https://www.figma.com/design/2UclqH1FR2GX32nGQyWq8R/Luxury-Bakery-Website-Design.

## Running the code

### Option 1: Using Docker (Recommended)

1. Make sure Docker and Docker Compose are installed on your system.

2. Build and start the container:
   ```bash
   docker-compose up --build
   ```

3. The website will be available at `http://localhost:3000`

4. To stop the container, press `Ctrl+C` or run:
   ```bash
   docker-compose down
   ```

### Option 2: Local Development

1. Run `npm i` to install the dependencies.

2. Run `npm run dev` to start the development server.

3. The website will be available at `http://localhost:3000`
